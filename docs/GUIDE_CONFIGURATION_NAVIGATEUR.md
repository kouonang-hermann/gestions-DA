# 🖥️ Guide de Configuration Navigateur - Affichage Optimal

## 📋 Introduction

Ce guide vous aide à configurer votre navigateur pour un **affichage optimal** de l'application de gestion des demandes, quel que soit votre PC ou votre résolution d'écran.

---

## ✅ Paramètres Recommandés

### 1. **Zoom du Navigateur**

Le zoom doit être à **100%** pour un affichage optimal.

#### Comment vérifier et ajuster :

**Google Chrome / Microsoft Edge :**
1. Appuyez sur `Ctrl + 0` (zéro) pour réinitialiser à 100%
2. Ou : Menu (⋮) → Zoom → Vérifier qu'il est à 100%

**Firefox :**
1. Appuyez sur `Ctrl + 0` (zéro)
2. Ou : Menu (☰) → Zoom → 100%

**Vérification visuelle :**
- L'URL doit afficher "100%" à côté de l'icône de zoom
- Les cartes du dashboard doivent être bien alignées
- Le texte doit être net et lisible

---

### 2. **Résolution d'Écran et Scaling Windows**

#### Paramètres Windows Recommandés :

**Pour écrans 1920x1080 (Full HD) :**
- Scaling : **100%** ou **125%**
- Résolution : **1920 x 1080**

**Pour écrans 1366x768 (Laptops standards) :**
- Scaling : **100%**
- Résolution : **1366 x 768**

**Pour écrans 2560x1440 (2K) :**
- Scaling : **125%** ou **150%**
- Résolution : **2560 x 1440**

#### Comment configurer :

1. **Clic droit sur le bureau** → Paramètres d'affichage
2. **Résolution** : Choisir la résolution native de votre écran
3. **Échelle** : Choisir 100%, 125% ou 150% selon votre confort
4. **Redémarrer le navigateur** après modification

---

### 3. **Accélération Matérielle**

L'accélération matérielle améliore le rendu graphique.

#### Google Chrome / Edge :

1. Menu (⋮) → **Paramètres**
2. **Système** (dans le menu de gauche)
3. **Activer "Utiliser l'accélération matérielle si disponible"**
4. **Relancer le navigateur**

#### Firefox :

1. Menu (☰) → **Paramètres**
2. **Général** → **Performances**
3. Décocher "Utiliser les paramètres de performance recommandés"
4. **Activer "Utiliser l'accélération matérielle si disponible"**
5. **Redémarrer Firefox**

---

### 4. **Paramètres d'Affichage du Navigateur**

#### Taille de Police par Défaut :

**Chrome / Edge :**
1. Paramètres → **Apparence**
2. **Taille de police** : Medium (16)
3. **Niveau de zoom de la page** : 100%

**Firefox :**
1. Paramètres → **Général** → **Langue et apparence**
2. **Taille** : 16
3. **Zoom par défaut** : 100%

## 🔧 Résolution des Problèmes Courants

### Problème 1 : Effet de Moiré / Lignes Ondulées

**Symptômes :**
- Lignes ondulées sur l'écran
- Effet arc-en-ciel sur les arrière-plans
- Texte flou

**Solutions :**

1. **Vérifier le zoom** : `Ctrl + 0`
2. **Désactiver le mode économie d'énergie** (Windows)
3. **Mettre à jour les pilotes graphiques** :
   - Clic droit sur le bureau → Paramètres d'affichage
   - Paramètres d'affichage avancés
   - Propriétés de la carte graphique
   - Mettre à jour le pilote

4. **Ajuster la fréquence de rafraîchissement** :
   - Paramètres d'affichage → Affichage avancé
   - Fréquence de rafraîchissement : 60 Hz minimum

---

### Problème 2 : Texte Flou ou Illisible

**Solutions :**

1. **Activer ClearType** (Windows) :
   - Rechercher "ClearType" dans Windows
   - Exécuter l'assistant ClearType
   - Suivre les étapes

2. **Vérifier le scaling Windows** :
   - Ne pas dépasser 150%
   - Privilégier 100% ou 125%

3. **Vider le cache du navigateur** :
   - Chrome/Edge : `Ctrl + Shift + Delete`
   - Cocher "Images et fichiers en cache"
   - Effacer les données

---

### Problème 3 : Cartes ou Tableaux Déformés

**Solutions :**

1. **Mode plein écran** : Appuyez sur `F11`
2. **Fermer les autres onglets** pour libérer de la mémoire
3. **Redémarrer le navigateur**
4. **Vérifier la résolution d'écran** (voir section 2)

---

### Problème 4 : Couleurs Délavées

**Solutions :**

1. **Vérifier le profil de couleur** :
   - Paramètres Windows → Système → Affichage
   - Paramètres d'affichage avancés
   - Gestion des couleurs
   - Utiliser le profil sRGB

2. **Désactiver le mode nuit** du navigateur
3. **Désactiver les extensions** qui modifient les couleurs

## 📊 Tests de Vérification

### Test 1 : Affichage du Dashboard

1. Connectez-vous à l'application
2. Accédez au dashboard
3. **Vérifiez :**
   - ✅ Les cartes sont bien alignées
   - ✅ Les chiffres sont nets et lisibles
   - ✅ Les couleurs sont vives (pas délavées)
   - ✅ Pas d'effet de moiré

### Test 2 : Modal de Préparation

1. Ouvrez une demande à préparer
2. Cliquez sur "Préparer"
3. **Vérifiez :**
   - ✅ Le tableau est lisible
   - ✅ Les colonnes sont bien alignées
   - ✅ Les inputs sont clairs
   - ✅ Pas de lignes ondulées

### Test 3 : Responsive

1. Réduisez la fenêtre du navigateur
2. **Vérifiez :**
   - ✅ Les éléments s'adaptent
   - ✅ Pas de débordement horizontal
   - ✅ Texte toujours lisible
